## Before starting

Make sure you have already installed:

- [npm](https://www.npmjs.com/) >= v6.14.5
- [NodeJS](https://nodejs.org/) >= v12.17.0

## Installation

Clone this repo.

Then, install dependencies:

```
npm i
```

## Run the app

Then, start the app:

```
npm start
```

Enjoy at http://localhost:3000/
